# Casulo AI Labs — Website

Static single-page website. No build step required.

## Files
- `index.html` — Main page
- `styles.css` — All styles
- `script.js` — Mobile nav, scroll animations, form handling

## Deploy

### Option 1: GitHub Pages
1. Push to a GitHub repo
2. Settings → Pages → Deploy from branch (main, /root)
3. Custom domain: add CNAME file with `casuloailabs.com`

### Option 2: Netlify
1. Drag & drop the folder at [app.netlify.com/drop](https://app.netlify.com/drop)
2. Set custom domain in site settings

### Option 3: Any static host
Upload the 3 files to any web server. No dependencies.

## Setup
- Replace `PLACEHOLDER` in the form action with your Formspree form ID
- Create a form at [formspree.io](https://formspree.io) and use the provided endpoint

## Fonts
Uses Google Fonts (Inter) — loaded via CDN. No local fonts needed.
